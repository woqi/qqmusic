# qqmusic
## 模块：

1. 推荐页面
2. 热门列表
3. 搜索歌曲
4. 播放器

## 要点：

1. 首页轮播
2. Tab 组件 
3. 电台热门列表
4. LazyLoad - 节流(throttle) IntersectionObserver
5. 分析接口 - 伪造请求 request request-promise
6. 搜索滚动加载
7. 歌词解析 - 正则 歌词转义
8. 播放器制作
9. ES6 模块 - Webpack Babel
